Python

My most used and loved programming language. Python provides minimalistic syntax which allows for writing quick prototypes/scripts. Most importantly allows developers to automate tedious tasks with ease thanks to the high amounts of Python libraries!

Java

My 2nd most used language. I often find myself using Java when I want to do any projects related to Web APIs or game servers. The language itself is very Object-Oriented however recently I feel that Kotlin and C# would be a better candidate for what I do with Java most of the time.

C++ | C# (Unity)

C++ is the first language I learned and is being taught in my first year at University. C# Unity is a language I picked up when starting my game development journey. Overall these are my weakest languages, but I will always find myself using it when it comes to game development as the two biggest engines (UE4, Unity) utilize them.

Projects

All of my biggest and most fun projects.

SwordieDB

SwordieDB is a pip-compatible MapleStory database API that lets users create database-related products with ease.

SpiritIDAPlugin

An IDA (Interactive Disassembler) Plugin that adds some QOL features for MapleStory client reverse engineering.

Lapis

Lapis is a fully plug-n-play MapleStory Discord bot that communicates with the database to relay some in-game information.

Forward

A 2D Platformer game with a teleport mechanic, a game (Unity) created in a 1 week game jam under the theme "Holes".

BattleAnalysis176

A C++ Program that read's process memory to calculate how much in-game currency you would make after a certain amount of time.

ProgrammingTriviaBot

A Discord Bot that allows trivia-style games to played by multiple players at a time. With a built in database-system using only JSON data.
